

# GetWebResources

>  一键保存网站的所有资源到本地

## 功能

- 按照 域名 与 资源类型  分类文件夹. 

- 可以设置多个域名筛选, 仅获取对应域名里的资源

- 资源默认保存在 out 目录中



## 小技巧
全局安装  `npm i -g prettier` 

在目录中执行 `prettier --write .` (最后有个点)

可以自动递归子文件夹,格式化所有的前端代码文件



## 环境

> Windows 11
>
> Microsoft Visual Studio 2022
>
> WPF
>
> .Net 6
