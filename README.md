

# GetWebResources

>  一键保存网站的所有资源到本地

## 功能

> 按照 域名 与 资源类型  分类文件夹. 
>
> 可以设置多个域名筛选, 仅获取对应域名里的资源
>
> 资源默认保存在 out 目录中

## 环境

> Windows 10 企业版 版本号 1909
>
> Microsoft Visual Studio 2019 版本 16.9.1
>
> WPF
>
> .net 5.0.4
